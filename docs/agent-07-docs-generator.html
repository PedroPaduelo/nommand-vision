Documentação

Gera documentação automática de código, APIs e componentes.

📋 Descrição

Gera documentação automática de código, APIs e componentes.

🎯 Responsabilidades

  • Extrair docstrings e comentários
  • Gerar API docs (OpenAPI/Swagger)
  • Documentar componentes (props, exemplos)
  • Criar guias de uso passo a passo
  • Publicar em formato HTML/Markdown

🔧 Tecnologias

  • TypeDoc
  • JSDoc
  • Swagger
  • Storybook
  • DocFX

⚡ Como Usar

  • Acesse a página Agentes
  • Selecione Docs Generator
  • Forneça input ou contexto
  • Execute e aguarde resultado

🔄 Fluxo de Execução

  1. Analisa repositório ou pasta src/: Analisa repositório ou pasta src/
  2. Extrai tipos, interfaces, comentários: Extrai tipos, interfaces, comentários
  3. Gera estrutura de documentação: Gera estrutura de documentação
  4. Exporta HTML, Markdown ou PDF: Exporta HTML, Markdown ou PDF
  5. Publica em docs/ ou site: Publica em docs/ ou site

📡 API Endpoint

POST /api/agents/:id/run

Exemplo de requisição:

{ "input": "generate API docs", "projectId": "uuid" }

📊 Dados Coletados

  • Entrada: Prompt, arquivos, contexto do projeto
  • Saída: Resultado processado (código, relatório, etc.)
  • Métricas: Total de execuções, taxa de sucesso, tempo médio, tokens usados
  • Logs: Histórico completo de cada execução
