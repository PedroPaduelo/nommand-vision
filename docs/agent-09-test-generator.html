Test Generator

Testes

Gera testes unitários, integração e E2E automaticamente a partir do código.

📋 Descrição

Gera testes unitários, integração e E2E automaticamente a partir do código.

🎯 Responsabilidades

  • Gerar skeleton de testes unitários
  • Criar mocks e stubs para dependências
  • Escrever testes de integração
  • Gerar testes E2E com cenários comuns
  • Atingir cobertura >80%

🔧 Tecnologias

  • Jest
  • Vitest
  • Testing Library
  • Playwright
  • Cypress
  • Sinon

⚡ Como Usar

  • Acesse a página Agentes
  • Selecione Test Generator
  • Forneça input ou contexto
  • Execute e aguarde resultado

🔄 Fluxo de Execução

  1. Recebe função, componente ou endpoint: Recebe função, componente ou endpoint
  2. Analisa assinatura e lógica: Analisa assinatura e lógica
  3. Gera testes para casos comuns: Gera testes para casos comuns
  4. Adiciona mocks de dependências: Adiciona mocks de dependências
  5. Executa e verifica cobertura: Executa e verifica cobertura

📡 API Endpoint

POST /api/agents/:id/run

Exemplo de requisição:

{ "input": "generate tests for UserService", "projectId": "uuid" }

📊 Dados Coletados

  • Entrada: Prompt, arquivos, contexto do projeto
  • Saída: Resultado processado (código, relatório, etc.)
  • Métricas: Total de execuções, taxa de sucesso, tempo médio, tokens usados
  • Logs: Histórico completo de cada execução
