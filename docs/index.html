Pontos de Risco

Análise de riscos e mitigações do sistema.

ALTO

Rate Limit Insuficiente

Auth endpoints têm rate limit por IP, mas workspaces compartilham limites.

Mitigação: Rate limit por usuário + workspace
ALTO

JWT sem Revogação

Refresh tokens estão em DB mas access tokens não têm blacklist.

Mitigação: Token blacklist ou short expiry
MÉDIO

SQL Injection via JSON

Drizzle ORM protege, mas queries dinâmicas podem ser vulneráveis.

Mitigação: Zod validation + prepared statements
MÉDIO

Leak de Dados Cross-Workspace

Middleware auth verifica workspaceId, mas queries podem falhar.

Mitigação: Audit de queries + RLS
MÉDIO

Agent Prompt Injection

Usuário controla systemPrompt em agentes customizados.

Mitigação: Sandbox + review de prompts
MÉDIO

Custos de IA Descontrolados

Sem limites, agentes podem gerar custos excessivos.

Mitigação: Budget limits + alertas
BAIXO

Logs Sensíveis

Logs podem conter tokens ou dados pessoais.

Mitigação: Auto-sanitization + redaction
BAIXO

Deploy Falhos Silenciosos

Deploys podem falhar sem notificação adequada.

Mitigação: Notificações obrigatórias
