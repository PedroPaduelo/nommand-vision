Analytics

Gera relatórios, dashboards e insights sobre uso do sistema e comportamento dos usuários.

📋 Descrição

Gera relatórios, dashboards e insights sobre uso do sistema e comportamento dos usuários.

🎯 Responsabilidades

  • Coletar métricas de uso (pageviews, events)
  • Calcular custos de tokens e recursos
  • Gerar timeseries de activity
  • Criar painéis em tempo real
  • Alertar sobre anomalias

🔧 Tecnologias

  • PostgreSQL analytics
  • Grafana
  • Metabase
  • ClickHouse
  • TimescaleDB

⚡ Como Usar

  • Acesse a página Agentes
  • Selecione Analytics
  • Forneça input ou contexto
  • Execute e aguarde resultado

🔄 Fluxo de Execução

  1. Consulta dados agregados do banco: Consulta dados agregados do banco
  2. Processa métricas (tokens, custos, execuções): Processa métricas (tokens, custos, execuções)
  3. Gera gráficos e tabelas: Gera gráficos e tabelas
  4. Envia relatórios por email: Envia relatórios por email
  5. Atualiza cache de dashboards: Atualiza cache de dashboards

📡 API Endpoint

POST /api/agents/:id/run

Exemplo de requisição:

{ "input": "relatório de uso de agentes na última semana", "projectId": "uuid" }

📊 Dados Coletados

  • Entrada: Prompt, arquivos, contexto do projeto
  • Saída: Resultado processado (código, relatório, etc.)
  • Métricas: Total de execuções, taxa de sucesso, tempo médio, tokens usados
  • Logs: Histórico completo de cada execução
