WebSocket Server

O sistema usa WebSocket (lib ws) para comunicação em tempo real. Conexão é autenticada via JWT token na query string.

ws://dominio/ws?token=<access_token>

Fluxo de Conexão

Cliente conecta WS
↓
Server recebe upgrade
↓
Extrai token da query
↓
verifyAccessToken()
↓
Se válido: aceita conexão
↓
ws.on('connection')
↓
Adiciona ao wsManager
↓
Envía 'connected'

WS Connection Manager 

// ws-manager.ts
class WSManager {
  connections: Map<userId, WebSocket[]>

  addConnection(userId: string, ws: WebSocket) {
    const list = this.connections.get(userId) || [];
    list.push(ws);
    this.connections.set(userId, list);
  }

  removeConnection(userId: string, ws: WebSocket) {
    const list = this.connections.get(userId);
    if (list) {
      const filtered = list.filter(connection => connection !== ws);
      if (filtered.length === 0) {
        this.connections.delete(userId);
      } else {
        this.connections.set(userId, filtered);
      }
    }
  }

  sendToUser(userId: string, event: string, data: any) {
    const connections = this.connections.get(userId);
    if (connections) {
      connections.forEach(ws => {
        ws.send(JSON.stringify({ event, data }));
      });
    }
  }
}

// Uso:
wsManager.sendToUser(userId, 'notification:new', notification);

Heartbeat & Reconexão

O servidor envia ping a cada 30 segundos. Clientes devem responder com pong. Conexões inativas são removidas automaticamente. 

// Servidor (ws.ts):
const HEARTBEAT_INTERVAL = 30_000;

setInterval(() => {
  wss.clients.forEach(ws => {
    if (!ws.isAlive) {
      ws.terminate(); // conexão morta
      return;
    }
    ws.isAlive = false;
    ws.ping();
  });
}, HEARTBEAT_INTERVAL);

ws.on('pong', () => {
  ws.isAlive = true;
});

// Cliente (React):
const ws = new WebSocket(url);
ws.on('pong', () => {
  // conexão viva
});
ws.on('close', () => {
  // tentar reconnectar após delay
});

Hook useWebSocket 

// Hook customizado (exemplo):
function useWebSocket(token: string) {
  const [isConnected, setIsConnected] = useState(false);
  const [messages, setMessages] = useState<WSMessage[]>([]);

  useEffect(() => {
    const ws = new WebSocket(`${WS_URL}?token=${token}`);

    ws.onopen = () => setIsConnected(true);
    ws.onclose = () => setIsConnected(false);
    ws.onmessage = (event) => {
      const msg = JSON.parse(event.data);
      setMessages(prev => [...prev, msg]);
    };

    return () => ws.close();
  }, [token]);

  const send = (event: string, data: any) => {
    ws.send(JSON.stringify({ event, data }));
  };

  return { isConnected, messages, send };
}
