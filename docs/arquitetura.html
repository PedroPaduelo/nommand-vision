Fluxo de Requisição

Browser → API Request
↓
CORS Middleware (valida origem)
↓
Request ID Middleware (tracking)
↓
Auth Middleware (verifica JWT)
↓
Route Handler (valida Zod)
↓
Service Layer (lógica)
↓
Database (Drizzle ORM)
↓
Response JSON
