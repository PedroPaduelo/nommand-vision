Desenvolvimento

📋 Descrição

Escreve código de alta qualidade a partir de especificações e requisitos. Suporta múltiplas linguagens e frameworks.

🎯 Responsabilidades

  • Gerar código limpo e manutenível
  • Seguir padrões do projeto (ESLint, Prettier)
  • Criar testes unitários junto com o código
  • Documentar funções e componentes
  • Refatorar quando necessário

🔧 Tecnologias

  • TypeScript
  • JavaScript
  • React
  • Vue
  • Node.js
  • Python
  • SQL

⚡ Como Usar

  • Acesse a página Agentes
  • Selecione CodeWriter
  • Forneça input ou contexto
  • Execute e aguarde resultado

🔄 Fluxo de Execução

  1. Recebe especificação em linguagem natural: Recebe especificação em linguagem natural
  2. Analisa contexto do projeto (arquivos existentes): Analisa contexto do projeto (arquivos existentes)
  3. Gera código + testes + tipagem: Gera código + testes + tipagem
  4. Valida sintaxe e formatação: Valida sintaxe e formatação
  5. Retorna arquivos prontos para commit: Retorna arquivos prontos para commit

📡 API Endpoint

POST /api/agents/:id/run

Exemplo de requisição:

{ "input": "Crie uma API REST para usuários com autenticação JWT", "projectId": "uuid" }

📊 Dados Coletados

  • Entrada: Prompt, arquivos, contexto do projeto
  • Saída: Resultado processado (código, relatório, etc.)
  • Métricas: Total de execuções, taxa de sucesso, tempo médio, tokens usados
  • Logs: Histórico completo de cada execução
