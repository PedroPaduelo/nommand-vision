📋 Descrição

Monitora saúde do sistema, logs e dispara alertas automáticos em caso de anomalias.

🎯 Responsabilidades

  • Health check de serviços (DB, Redis, APIs)
  • Análise de logs (errors, warnings)
  • Detecção de anomalias (spike de erros, latência)
  • Disparar alertas (Slack, Email, SMS)
  • Dashboard de status em tempo real

🔧 Tecnologias

  • Prometheus
  • Grafana
  • Sentry
  • Datadog
  • UptimeRobot

⚡ Como Usar

  • Acesse a página Agentes
  • Selecione Monitor
  • Forneça input ou contexto
  • Execute e aguarde resultado

🔄 Fluxo de Execução

  1. Coleta métricas do sistema: Coleta métricas do sistema
  2. Verifica thresholds definidos: Verifica thresholds definidos
  3. Detecta anomalias com ML básico: Detecta anomalias com ML básico
  4. Classifica severidade: Classifica severidade
  5. Envia notificação via Inbox: Envia notificação via Inbox

📡 API Endpoint

POST /api/agents/:id/run

Exemplo de requisição:

{ "input": "monitorar API latency", "projectId": "uuid" }

📊 Dados Coletados

  • Entrada: Prompt, arquivos, contexto do projeto
  • Saída: Resultado processado (código, relatório, etc.)
  • Métricas: Total de execuções, taxa de sucesso, tempo médio, tokens usados
  • Logs: Histórico completo de cada execução
