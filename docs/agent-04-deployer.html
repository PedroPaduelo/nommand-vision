Deploy

Automatiza o processo de deploy para staging e produção com CI/CD completo.

📋 Descrição

Automatiza o processo de deploy para staging e produção com CI/CD completo.

🎯 Responsabilidades

  • Build de artifacts (bundle, minificação)
  • Executar testes automatizados
  • Deploy multi-ambiente (preview, staging, prod)
  • Health check pós-deploy
  • Rollback automático em falha

🔧 Tecnologias

  • Docker
  • EasyPanel
  • GitHub Actions
  • Shell scripts
  • Kubernetes

⚡ Como Usar

  • Acesse a página Agentes
  • Selecione Deployer
  • Forneça input ou contexto
  • Execute e aguarde resultado

🔄 Fluxo de Execução

  1. Recebe trigger (push, webhook, botão): Recebe trigger (push, webhook, botão)
  2. Instala dependencies (npm ci): Instala dependencies (npm ci)
  3. Roda testes e build: Roda testes e build
  4. Stream de logs em tempo real: Stream de logs em tempo real
  5. Deploy + health check: Deploy + health check
  6. Atualiza status no banco: Atualiza status no banco

📡 API Endpoint

POST /api/deploys

Exemplo de requisição:

{ "projectId": "uuid", "environment": "production", "branch": "main" }

📊 Dados Coletados

  • Entrada: Prompt, arquivos, contexto do projeto
  • Saída: Resultado processado (código, relatório, etc.)
  • Métricas: Total de execuções, taxa de sucesso, tempo médio, tokens usados
  • Logs: Histórico completo de cada execução
