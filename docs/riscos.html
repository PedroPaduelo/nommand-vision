Riscos de Segurança

Crítico

SQL Injection

Uso de Drizzle ORM com parameterized queries previne SQL injection. Mitigado via ORM. 

// Seguro:
db.select().from(users).where(eq(users.id, userId));
// Parâmetros são escapados automaticamente
Alto

Rate Limit Bypass

Implementado rate limit multi-camada (IP + Workspace + User). Stores em memória podem ser perdidos em restart. Mitigado com stores persistentes planejados.

Médio

JWT Token Theft

Tokens são armazenados em memória apenas (não localStorage). Implementada blacklist para revogação imediata. Recomendado: Implementar refresh rotation.

Baixo

XSS via User Input

Sanitize-html usado em serviços relevantes. Frontend usa React (auto-escaping). Verificar sanitização em todos os pontos de entrada.

Riscos de Disponibilidade

Alto

Single Point of Failure - Banco

Se o PostgreSQL cair, todo o sistema para. Recomendado: Implementar replicação e connection pooling robusto. Já há pool configurado no drizzle.

Médio

Memory Leak - Logs em Memória

Deploy logs armazenados em memória podem crescer indefinidamente. Implementar limpeza automática após X horas. Mitigado: Rotinas de cleanup planejadas.

Baixo

WebSocket Server Scalling

Atualmente, conexões WS são mantidas em memória (Map). Para múltiplas instâncias, é necessário adapter Redis.

Tratamento de Erros 

// middleware/error-handler.ts
app.use('*', async (c, next) => {
  try {
    await next();
  } catch (err) {
    console.error('Request error:', err);

    // Erros de validação (Zod)
    if (err instanceof ZodError) {
      return c.json({ error: { code: 'VALIDATION', details: err.errors } }, 400);
    }

    // Erros de negócio (com error.code)
    if (err.message && err.code) {
      return c.json({ error: { code: err.code, message: err.message } }, getStatusForCode(err.code));
    }

    // Erro genérico (não expor stack trace em prod)
    if (env.NODE_ENV === 'production') {
      return c.json({ error: { code: 'INTERNAL_ERROR', message: 'Something went wrong' } }, 500);
    }

    // Dev: mostra stack trace
    return c.json({ error: { code: 'INTERNAL_ERROR', message: err.message, stack: err.stack } }, 500);
  }
});
