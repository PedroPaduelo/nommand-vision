📋 Descrição

Executa testes automatizados, identifica bugs e valida funcionalidades antes do deploy.

🎯 Responsabilidades

  • Testes unitários (funções isoladas)
  • Testes de integração (API + banco)
  • Testes E2E (fluxo completo)
  • Validação de regressão
  • Relatórios de cobertura

🔧 Tecnologias

  • Jest
  • Vitest
  • Playwright
  • Cypress
  • Puppeteer
  • Supertest

⚡ Como Usar

  • Acesse a página Agentes
  • Selecione QA Tester
  • Forneça input ou contexto
  • Execute e aguarde resultado

🔄 Fluxo de Execução

  1. Recebe comando para testar feature ou projeto: Recebe comando para testar feature ou projeto
  2. Executa suite de testes configurada: Executa suite de testes configurada
  3. Captura falhas e screenshots: Captura falhas e screenshots
  4. Gera relatório detalhado: Gera relatório detalhado
  5. Notifica equipe em caso de falha crítica: Notifica equipe em caso de falha crítica

📡 API Endpoint

POST /api/agents/:id/run

Exemplo de requisição:

{ "input": "testar checkout flow", "projectId": "uuid" }

📊 Dados Coletados

  • Entrada: Prompt, arquivos, contexto do projeto
  • Saída: Resultado processado (código, relatório, etc.)
  • Métricas: Total de execuções, taxa de sucesso, tempo médio, tokens usados
  • Logs: Histórico completo de cada execução
